# Rattle: linting framework with auto-fixes

Rattle is a LibCST-based Python linter with autofixes, configurable built-in rules, and support for custom in-repo rules.

If you want the package-facing overview, installation, and first commands, start
with the repository README. This docs home focuses on where to go next.
